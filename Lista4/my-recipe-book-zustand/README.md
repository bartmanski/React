# My Recipe Book - Zustand

A React + Zustand recipe book application with the ability to add, delete, search, and manage favorite recipes.

## Features

- Display recipes with titles and descriptions
- Add new recipes with a simple form
- Delete recipes
- Mark recipes as favorites (visually indicated)
- Filter to show only favorites
- Search recipes by keyword (searches in title and description)
- Built with TypeScript for type safety
- Zustand for state management

## Installation

```bash
npm install
```
